Archives

It used to be possible to view all blog articles from a certain day, month and year. Like Chris Roos, I don’t think there’s any value in these pages so I’ve got rid of them. If you really want to find everything I posted on a given day, month or year then you can use Google with the site modifier e.g. searching for site:jamesmead.org/blog/2009-12 would give you all posts from December 2009.
